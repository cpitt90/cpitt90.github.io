# Chris Pitt — UX Design Portfolio

Personal portfolio site for Chris Pitt, Senior UX Designer (Fintech).

## Pages
- `index.html` — Home / selected work
- `about.html` — About
- `work-manager-hub-broadsheet.html` — Case study: Work Manager Hub
- `fund-finance-ratings.html` — Case study: Fund Finance Ratings

## Stack
Static HTML/CSS/JS. No build step — open any page in a browser.

## Local preview
```bash
python3 -m http.server 8000
```
Then visit http://localhost:8000

## Deployment
Hosted on GitHub Pages from the `main` branch. Every push auto-deploys.
